Bu projem c dilinde kendimi geliştirmek için yapmış olduğum basit bir dungeon oyunudur

Oyunda 3 karakterden birini seçerek başlıyorsunuz. Her karakterin kendine göre özellikleri var.
Karakter seçtikten sonra oyun sizden 3 farklı yoldan birini seçip gitmenizi istiyor.
Her yolda farklı düşmanlarla karşılaşıp savaşıyorsunuz.
Bu oyunun kodlamasında hatalarım fazla olabilir. Eğer daha temiz bir kod ile yazılmış şekilde düzenlerseniz bana da göstermeyi unutmayın :)


Hello !!!!

This project is a simple dungeon game I made to improve myself in c language.

You start the game by choosing one of 3 characters. Each character has their own characteristics.
After choosing a character, the game asks you to choose one of 3 different ways and go.
You encounter and fight different enemies on each road.
I may have a lot of mistakes in the coding of this game. If you edit it with a cleaner code, don't forget to show it to me :)
